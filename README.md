# UI тесты
# Тестирование веб-приложения https://stellarburgers.education-services.ru/
Тесты в папке `tests`
Отдельные классы с Page Object для страниц в директории `pages`
## Проект
### Тесты
- Проверяет переход в Конструктор, переход в Ленту заказов, появление модального окна, закрытие модального окна, счётчик индегриентов (`test_main_function`)
- Проверяет увеличение счетчика 'Выполнено за всё время', увеличение счетчика 'Выполнено за сегодня', заказ в разделе 'В работе' (`test_order_feed`)
## Запустить тесты 
pytest -v 
## Установить зависимости 
pip install -r requirements.txt
